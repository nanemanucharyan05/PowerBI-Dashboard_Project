# Power BI Sales Dashboard

## Overview
This project is an interactive sales dashboard built using Power BI.  
It analyzes sales, profit, and product performance to generate business insights.

---

## Dashboard Features

- Total Sales, Profit, and Quantity KPIs
- Sales trend over time (Year-wise)
- Profit trend analysis
- Sales by Category and Sub-Category
- Regional distribution of sales
- Interactive filters (Region, Category, Year)

---

## Data Preparation

The dataset was cleaned using Power Query:
- Removed null values
- Fixed data types
- Removed duplicates
- Created calculated columns (Year, etc.)

---

## Key Insights

- Sales vary significantly across categories
- Some products generate high sales but low profit
- Profit trends do not always follow sales trends
- Regional performance differs across markets

---

## Tools Used
- Power BI
- Power Query
- DAX
